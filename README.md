## 🚀 Overview
**Secura** is a robust Windows cybersecurity desktop application designed to provide dual-layer file security. It combines the lightning speed of **NTFS ACL Fast Locking** with the unbreakable security of **AES-256-GCM Deep Encryption** in one unified, zero-telemetry vault.

## ✨ Key Features
- **⚡ Fast Lock:** Instant access restriction using Windows NTFS Access Control Lists (ACLs).
- **🔒 Deep Encryption:** Military-grade AES-256-GCM cryptographic protection for absolute confidentiality.
- **☁️ Cloud Recovery:** Securely sync your encrypted vault to Google Drive for disaster recovery.
- **🛡️ Tamper-Proof Logs:** Cryptographically hashed security logs to detect unauthorized activity.

## 📥 Download & Installation
1. Navigate to the **[Releases](../../releases/latest)** section of this repository.
2. Download the latest `Secura_Windows_Installer.exe`.
3. Run the installer and follow the standard Windows setup wizard.

> **Note:** As this software modifies system-level ACLs, Windows Defender/SmartScreen may show a prompt. Click **More info** -> **Run anyway** to proceed with the installation safely.
